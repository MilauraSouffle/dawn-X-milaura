## Actions à faire dans l'admin Shopify

### 1. Configurer les policies

1. Se connecter à Shopify Admin
2. Aller dans **Settings > Policies**
3. Copier-coller le contenu ci-dessus dans chaque policy
4. **Remplacer les placeholders** :
   - `[DATE]` → Date actuelle
   - `[NOM ENTREPRISE]` → Raison sociale
   - `[SIRET]` → Numéro SIRET
   - `[ADRESSE]` → Adresse postale complète
   - `[EMAIL]` → Email de contact
   - `[TÉLÉPHONE]` → Numéro de téléphone
   - `[MÉDIATEUR]` → Organisme de médiation choisi
5. Cliquer sur **Save**

### 2. Créer les menus footer

1. Aller dans **Online Store > Navigation**
2. Créer un menu nommé `footer` (si pas existant)
3. Ajouter les liens :
   - Collections > Bougies Émotionnelles
   - Collections > Bijoux Énergétiques
   - Pages > Quiz Émotionnel
   - Pages > À propos
   - Pages > Contact

3. Créer un second menu `footer-legal` avec :
   - Politique de confidentialité (lien auto)
   - Conditions générales (lien auto)
   - Politique de remboursement (lien auto)

### 3. Configurer les réseaux sociaux

1. Aller dans **Settings > Social media**
2. Ajouter les liens Instagram, Facebook, TikTok, Pinterest

### 4. Tester

1. Aller sur le site
2. Vérifier que le footer s'affiche correctement
3. Cliquer sur chaque lien du footer
4. Tester le formulaire newsletter
